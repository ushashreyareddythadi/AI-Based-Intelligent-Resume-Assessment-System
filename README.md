# 🤖 AI-Based Intelligent Resume Assessment System

🚀 **Live Demo:** https://ai-based-intelligent-resume-assessm.vercel.app/

An AI-powered web application that analyzes resumes, evaluates ATS compatibility, identifies skill gaps, and provides intelligent recommendations to help users improve their resumes for better job opportunities.

---

## 📖 About the Project

The **AI-Based Intelligent Resume Assessment System** is designed to assist job seekers in creating stronger, ATS-friendly resumes. Users can upload their resumes, receive AI-powered feedback, analyze skills, and identify areas for improvement.

The platform provides an intuitive and responsive interface while leveraging AI and cloud technologies to deliver personalized insights.

---

## ✨ Features

- 📄 Upload and analyze resumes
- 🤖 AI-powered resume assessment
- 📊 ATS compatibility evaluation
- 🎯 Skill extraction and analysis
- 📈 Skill gap identification
- 💡 Personalized improvement suggestions
- 🔍 Resume quality analysis
- ☁️ Supabase integration for backend services
- 📱 Responsive and user-friendly interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ushashreyareddythadi/AI-Based-Intelligent-Resume-Assessment-System.git
```

Navigate to the project folder:

```bash
cd ai-resume-analyzer
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:8080
```
